================================================================================
                         TROUBLESHOOTING
================================================================================

Problem: Backend won't start
Solution: Make sure ports 8000 is free
          Close any other Python processes

Problem: "Module not found" errors
Solution: Run: pip install -r requirements.txt

Problem: Dashboard shows "Backend Offline"
Solution: Make sure run_backend.bat is running in another terminal

Problem: PyTesseract error
Solution: Install Tesseract OCR:
          https://github.com/UB-Mannheim/tesseract/wiki
          (Not required for other 3 modules to work)

================================================================================
                     WHAT EACH MODULE DOES
================================================================================

Module A (Tender-Watch):
  Uses: Sentence Transformers (all-MiniLM-L6-v2)
  Input: Multiple PDF files
  Output: Similarity scores between documents
  Flags: Pairs with >90% similarity

Module B (Price-Guard):
  Uses: PyTesseract OCR + Regex parsing
  Input: Invoice image (PNG/JPG)
  Output: Extracted items and prices vs market rates
  Flags: Items >50% above market price

Module C (Ghost-Hunter):
  Uses: NetworkX graph analysis
  Input: Payroll CSV
  Output: Employee clusters sharing metadata
  Flags: Clusters with >2 connected employees

Module D (Welfare-Shield):
  Uses: RapidFuzz fuzzy string matching
  Input: Pension list + Death registry CSVs
  Output: Matched deceased beneficiaries
  Flags: Matches with >85% similarity score
