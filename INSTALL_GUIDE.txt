================================================================================
TROUBLESHOOTING COMMON ISSUES
================================================================================

Issue 1: "ModuleNotFoundError: No module named 'pdfplumber'"
-------------------------------------------------------------
Solution: Dependencies not installed yet
Run: python install.py


Issue 2: "pytesseract.pytesseract.TesseractNotFoundError"
----------------------------------------------------------
Solution: Tesseract OCR not installed (separate from Python)
Download: https://github.com/UB-Mannheim/tesseract/wiki
Install the .exe file, then add to PATH


Issue 3: "torch installation fails"
------------------------------------
Solution: Install CPU version manually:
pip install torch --index-url https://download.pytorch.org/whl/cpu


Issue 4: "Streamlit won't start"
---------------------------------
Solution: Check if dependencies are installed:
python -c "import streamlit; print('OK')"

If error, reinstall:
pip install streamlit==1.28.0


Issue 5: "Port 8501 already in use"
------------------------------------
Solution: Kill existing Streamlit process or use different port:
streamlit run streamlit_app.py --server.port 8502


Issue 6: "transformers/DistilBERT download fails"
--------------------------------------------------
Solution: First run downloads ~250MB model - needs internet
Wait for download to complete (2-3 minutes)
