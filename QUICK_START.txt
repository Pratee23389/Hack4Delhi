üÜò TROUBLESHOOTING:
================================================================================

Problem: "setup.bat fails"
Solution: Ensure Python 3.8+ is installed and added to PATH

Problem: "Module not found" errors
Solution: Activate virtual environment: venv\Scripts\activate.bat

Problem: "PyTesseract not found"
Solution: Install Tesseract OCR from:
          https://github.com/UB-Mannheim/tesseract/wiki

Problem: "Torch installation fails"
Solution: pip install torch --index-url https://download.pytorch.org/whl/cpu

Problem: Dashboard doesn't open
Solution: Manually open browser to http://localhost:8501
